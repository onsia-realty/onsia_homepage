# 랜딩페이지 솔라피(Solapi) 문자 발송 매뉴얼

## 개요

랜딩페이지에서 관심고객 등록(문의) 시 관리자에게 SMS 알림을 자동 발송하는 시스템.

```
관심고객 등록 → DB 저장 (Supabase) → SMS 발송 (Solapi) → 관리자 수신
```

---

## 관련 파일

| 파일 | 역할 |
|------|------|
| `src/lib/solapi.ts` | Solapi SDK를 이용한 SMS 발송 모듈 |
| `src/app/api/landing-inquiry/route.ts` | 문의 등록 API (DB 저장 + SMS 발송) |
| `src/app/(landing)/[slug]/InquiryForm.tsx` | 관심고객 등록 폼 (프론트엔드) |

### 주의사항

- **줄바꿈/공백 금지**: 환경변수 값 끝에 줄바꿈(`\n`)이나 공백이 들어가면 SMS 발송 실패
- `printf`를 사용하면 줄바꿈 없이 값 입력 가능 (`echo`는 줄바꿈 추가됨)
- 코드에서 `.trim().replace(/[\r\n]/g, '')` 처리로 방어하고 있지만, 환경변수 자체를 깨끗하게 설정하는 것이 좋음

## 문자 발송 흐름

### 1. 프론트엔드 (InquiryForm.tsx)

```
사용자 입력 (이름, 연락처) → 전화번호 자동 포맷팅 (010-XXXX-XXXX)
→ 유효성 검증 → POST /api/landing-inquiry
```

- 전화번호: `010-XXXX-XXXX` 자동 포맷팅
- 11자리 + `010` 시작 검증

### 2. API (landing-inquiry/route.ts)

```typescript
// 1. DB 저장 (Supabase - service_role로 RLS 우회)
const result = await submitLandingInquiry({ page_id, name, phone, ... })

// 2. SMS 발송 (await로 완료 보장)
const smsResult = await notifyAdmin({ name, phone, projectName, agentName })
```

- `await`로 SMS 발송 완료를 기다림 (Vercel Serverless 함수 종료 방지)
- SMS 실패해도 문의 등록은 성공 처리

### 3. SMS 발송 (solapi.ts)

```typescript
const messageService = new SolapiMessageService(API_KEY, API_SECRET)

await messageService.sendOne({
  to: ADMIN_PHONE,        // 수신번호
  from: SMS_SENDER_NUMBER, // 발신번호 (솔라피 등록 필수)
  text: '[온시아] 문의 접수...'
})
```

## 트러블슈팅

### 문자가 안 올 때 체크리스트

1. **솔라피 대시보드 → 발송 내역** 확인
   - "발송완료" → 수신 측 문제 (스팸 차단 등)
   - "발송불가" → 아래 사유 확인
   - 내역 없음 → API 호출 자체가 안 됨

2. **발송불가 사유별 대응**

| 에러 코드 | 사유 | 해결 |
|-----------|------|------|
| 1062 | 발신번호 미등록 | 솔라피에서 발신번호 등록 |
| ParseError | 전화번호 형식 오류 | 환경변수 줄바꿈/공백 제거 |
| NetworkError | API 키 형식 오류 | 환경변수 줄바꿈/공백 제거 |
| FailedToAddMessage | 잔액 부족 등 | 솔라피 충전 |

3. **Vercel 로그 확인**

```bash
# 실시간 로그 모니터링
vercel logs www.onsia.city

# 특정 배포 로그
vercel logs [deployment-url]
```

4. **환경변수 재설정** (줄바꿈 문제 해결)

```bash
vercel env rm SOLAPI_API_KEY production -y
printf "값" | vercel env add SOLAPI_API_KEY production
vercel --prod  # 재배포 필수
```

## 카카오 알림톡 전환 (향후)

솔라피에서 카카오 알림톡도 지원. 전환 시 필요한 것:

1. 카카오톡 비즈니스 채널 개설
2. 솔라피에 채널 연동
3. 메시지 템플릿 등록 + 심사 승인 (1~2영업일)
4. 코드에 `kakaoOptions` 추가

```typescript
await messageService.sendOne({
  to: ADMIN,
  from: SENDER,
  text,  // 알림톡 실패시 SMS 대체 발송
  kakaoOptions: {
    pfId: process.env.KAKAO_CHANNEL_ID!,
    templateId: process.env.KAKAO_TEMPLATE_ID!,
  }
})
```
