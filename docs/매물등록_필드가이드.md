## 매물 등록 필수 정보 (순번)

### ⭐ 필수 항목

| 순번 | 항목 | 설명 | 예시 |
|------|------|------|------|
| 1 | **현장명** | 분양 현장 이름 | `백운호수 푸르지오 숲속의 아침` |
| 2 | **위치** | 시/도 + 시/군/구 + 상세주소 | `경기도 의왕시 학의동 654번지` |
| 3 | **분양가** | 대표 평형 기준 (억 단위) | `16억` |
| 4 | **계약금** | (억 단위) | `0.8억 (5%)` |
| 5 | **준공예정일** | YYYY-MM-DD | `2027-06-30` |
| 6 | **총 세대수** | 전체 호실 수 | `842세대` |
| 7 | **건물 유형** | APARTMENT/OFFICETEL/VILLA/COMMERCIAL | `OFFICETEL` |

### 📋 권장 항목

| 순번 | 항목 | 설명 | 예시 |
|------|------|------|------|
| 8 | **시행사** | 없으면 "시행사 미정" | `엠디엠플러스` |
| 9 | **시공사** | 건설사 이름 | `대우건설(푸르지오)` |
| 10 | **평형 구성** | 쉼표로 구분 | `99㎡(30평), 119㎡(36평)` |
| 11 | **특장점** | 핵심 셀링포인트 (한 줄) | `백운호수 숲세권, GTX-C 예정` |
| 12 | **위치 설명** | 교통, 주변 인프라 | `강남 20분대, 판교 15분대` |

## 빌딩 타입 코드

| 코드 | 한글 |
|------|------|
| `APARTMENT` | 아파트 |
| `OFFICETEL` | 오피스텔 |
| `VILLA` | 빌라 |
| `TOWNHOUSE` | 타운하우스 |
| `COMMERCIAL` | 상가/지식산업센터 |

---

## 가격 입력 규칙

- **분양가/계약금**: 억 단위로 입력 → 시스템이 원화로 변환
  - 예: `5.5` 입력 → `550,000,000`원으로 저장
- **평단가**: 만원 단위로 입력
  - 예: `2500` 입력 → `25,000,000`원으로 저장
